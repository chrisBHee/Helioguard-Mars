# Helioguard-Mars

A Mars Solar Storm Early Warning System for the Challenger Center Space Challenge, leveraging Next.js for a user-friendly dashboard and integrating NASA’s Surya AI for real-time solar activity prediction. This project aims to provide astronauts and mission control with critical, timely alerts, enhancing safety and operational efficiency.

## Key Features:

*   User-friendly dashboard built with Next.js
*   Real-time solar activity prediction using NASA’s Surya AI
*   Timely alerts for astronauts and mission control
*   Enhanced safety and operational efficiency

## Technologies Used:

*   Next.js
*   NASA Surya AI
